Exploring Weather Trends, DAND Project 1:¶

In This Project, I'm going to use a Data about Alexandria city in Egypt, Contains yearly-average temperature of the City, Against the global temperature.¶

Data is extracted from two tables in a PostgreSql Database using sql-queries.
1- city_data - This contains the average temperatures for each city by year (ºC). 

    SELECT * 
    FROM city_data
    WHERE city = 'Alexandria' AND country = 'Egypt';

2- global_data - This contains the average global temperatures by year (ºC). 

    SELECT *
    FROM global_data;

Moving Average for the Global data and City data.¶

This is used to smooth out data and to make it easier to observe long term trends.

Rolling Average has been calculated to smooth out data and to make it easier to observe the trends when it be shown in Charts. It's been calculated for every 10 years to each single data but the first 10.
